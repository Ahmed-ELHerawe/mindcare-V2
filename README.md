# 🧠 MindCare — منصة الصحة النفسية

منصة React كاملة للصحة النفسية بـ ٣ أدوار مختلفة.

## 🚀 تشغيل المشروع

```bash
npm install
npm run dev
```

## 📱 صفحات المريض

| الصفحة | الوصف |
|--------|-------|
| الرئيسية | تسجيل المزاج + طبيبي + المواعيد القادمة |
| سجل المزاج | كل اللحظات المسجلة |
| محادثاتي | شات مع الطبيب + تسجيل صوتي |
| مواعيدي | حجز وإلغاء المواعيد |
| نصايح AI | نصايح بالمزاج + Claude API |
| الأطباء | استعراض الأطباء المتاحين |

## 🩺 صفحات الطبيب

| الصفحة | الوصف |
|--------|-------|
| الداشبورد | نظرة عامة على المرضى والمواعيد |
| مرضاي | سجل مزاج كل مريض + ملاحظات طبية |
| المحادثات | شات مع كل مريض |
| المواعيد | تأكيد ورفض المواعيد |
| التقارير | تقارير أسبوعية لكل مريض |

## 🛡️ صفحات الأدمن

| الصفحة | الوصف |
|--------|-------|
| الداشبورد | إحصائيات المنصة الكاملة |
| المستخدمون | كل المرضى والأطباء |

## 🛠️ التقنيات المستخدمة

- **React 18** + **Vite**
- **React Router v6** — التنقل بين الصفحات
- **Zustand + persist** — إدارة الحالة وحفظها
- **Formik + Yup** — فورم الحجز والتحقق
- **Claude API** — نصايح AI مخصصة
- **Web Speech API** — تسجيل صوتي في الشات
- **Mock Data** — بيانات تجريبية كاملة
